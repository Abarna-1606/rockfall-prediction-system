# AI-Based Rockfall Prediction and Alert System for Open-Pit Mines

## Project Description
A full-stack web application that predicts rockfall risk
in open-pit mines using AI-based weighted scoring of
sensor data and issues real-time color-coded alerts.

## Technology Stack
- Frontend: React.js
- Backend: Spring Boot (Java 21)
- Database: MySQL 8.0

## Modules
- Login and Registration
- Dashboard with live statistics
- Sensor Management
- AI Risk Prediction (GREEN / YELLOW / ORANGE / RED)
- Alert History

## How to Run

### Backend
cd backend
mvn spring-boot:run
Runs on http://localhost:8080

### Frontend
cd frontend/rockfall-ui
npm install
npm start
Runs on http://localhost:3000
